## Execução

O algorítmo está estruturado em 6 fases:

1. Instala as bibliotecas necessárias;
2. Importa as bibliotecas necessárias;
3. Gera Itens aleatórios;
4. Gera a função de apitidão do tipo MinMax e os individuos;
5. Inicializa a mochila dos indivíduos, a população e registra as funções
   que vai ser ultilizado pelo eaMuPlusLambda;
6. Gera e imprime dados estatísticos.

> Para uma execução sem problemas, recomendamos que execute do primeiro ao último tópico. Nos tópicos (5 e 6) está constituído em 4 blocos de código, para uma execução sem problemas execute-os em sequência (do primeiro ao último).
